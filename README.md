# Портфолио проектов по Data Science

В данном репозитории представлены проекты, выполненные в ходе обучения и практики в области Data Science и аналитики данных. Проекты расположены в порядке убывания сложности — от комплексных задач машинного обучения до базового анализа данных.

### Стек технологий

`Python` `Pandas` `NumPy` `Scikit-learn` `CatBoost` `LightGBM` `TensorFlow/Keras` `OpenCV` `NLTK` `Transformers` `Statsmodels` `Matplotlib` `Seaborn` `Plotly` `SQL` `Git`

## Список проектов

### 1. [Прогнозирование температуры стали для оптимизации производства](steel_bird.ipynb)
**Задача:** Построить модель для предсказания температуры стали на этапе обработки с целью снижения потребления электроэнергии.

**Описание:** Комплексный проект, включающий генерацию признаков на основе физики процесса, работу с временными рядами sensor data и построение регрессионной модели. Результаты предназначены для интеграции в систему управления технологическим процессом.

**Библиотеки:** `pandas`, `scikit-learn`, `lightgbm`, `catboost`, `numpy`, `matplotlib`, `seaborn`

**Выводы:** Разработана модель, предсказывающая температуру с MAE = X. Внедрение модели позволит оптимизировать энергозатраты и снизить себестоимость производства.

### 2. [Определение оптимального региона для добычи нефти](oil_samples.ipynb)
**Задача:** Построить модель для определения региона с максимальной прибылью от добычи и проанализировать риски с помощью Bootstrap.

**Описание:** Полный цикл задачи машинного обучения в бизнес-контексте: от построения модели регрессии до расчета экономических показателей и оценки рисков с применением статистических методов.

**Библиотеки:** `pandas`, `scikit-learn`, `numpy`, `bootstrap`

**Выводы:** На основе анализа рекомендован регион X, так как он имеет достаточный запас сырья и наименьшую вероятность убытков (<2.5%).

### 3. [Прогнозирование количества заказов такси на следующий час](taxi.ipynb)
**Задача:** Разработать модель для прогнозирования спроса на такси с требованием к качеству (RMSE < 48).

**Описание:** Проект по работе с временными рядами, включающий feature engineering (лаги, скользящие статистики), анализ трендов и сезонности, а также построение и валидацию прогнозных моделей.

**Библиотеки:** `pandas`, `scikit-learn`, `statsmodels`, `catboost`

**Выводы:** Целевое значение RMSE (48) достигнуто. Модель готова к внедрению для оптимизации работы водителей в пиковые периоды.

### 4. [Классификация токсичных комментариев](wikishop.ipynb)
**Задача:** Обучить модель для автоматической модерации комментариев со значением F1-меры не менее 0.75.

**Описание:** Проект по обработке естественного языка (NLP), включающий очистку текста, лемматизацию, векторизацию (TF-IDF) и построение моделей классификации с борьбой с дисбалансом классов.

**Библиотеки:** `pandas`, `scikit-learn`, `nltk`, `tf-idf`

**Выводы:** Модель на основе Logistic Regression показывает F1 = 0.85, что превышает требуемый порог и позволяет автоматизировать процесс модерации.

### 5. [Определение возраста по фотографии](photos_clients.ipynb)
**Задача:** Построить модель, которая по фотографии определит приблизительный возраст человека.

**Описание:** Проект в области компьютерного зрения с использованием сверточных нейронных сетей (CNN). Включает работу с большим набором изображений, аугментацию данных и transfer learning.

**Библиотеки:** `tensorflow/keras`, `pandas`, `numpy`

**Выводы:** Модель определяет возраст с MAE ~6-7 лет, что пригодно для задач анализа покупателей и контроля продаж.

**Библиотеки:** `pandas`, `scikit-learn`, `lightgbm`
